# app jsx

This repository contains the `zone-prev` Vite + React app with Tailwind CSS configured, and the `ZoneGardensApp.jsx` application source.

## Contents

- `zone-prev/`: Vite React app scaffold
- `zone-prev/src/App.jsx`: main application component
- `ZoneGardensApp.jsx`: original app source file

## Run locally

```bash
cd "zone-prev"
npm install
npm run dev
```

## Build

```bash
npm run build
```
